Alfresco Identity Service IDVERSION

This zip file includes a deployment of the Alfresco Identity Service.

Starting the Service

Windows

  • cd bin
  • .\standalone.bat

Linux

  • cd bin
  • ./standalone.sh

Admin Console

The Alfresco Identity Service is based upon Keycloak KVERSION

Once the service is started you can access the Keycloak Admin Console in order to setup the admin user and perform other administrative tasks

More Information

See the Alfresco Identity Service Project on GitHub for more information about the Alfresco Identity Service.
