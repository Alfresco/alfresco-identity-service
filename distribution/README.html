Alfresco Keycloak KVERSION

This zip file includes a deployment of the Keycloak with the Alfresco Realm and Alfresco Theme pre-installed.

Starting the Service

Windows

  • cd bin
  • .\kc.bat start --import-realm --http-relative-path=/auth --hostname=<HOSTNAME> --https-certificate-file=<PATH_TO_CERT_FILE> --https-certificate-key-file=<PATH_TO_CERT_KEY_FILE>

Linux

  • cd bin
  • ./kc.sh start --import-realm --http-relative-path="/auth" --hostname=<HOSTNAME> --https-certificate-file=<PATH_TO_CERT_FILE> --https-certificate-key-file=<PATH_TO_CERT_KEY_FILE>

Admin Console

This distribution is based upon Keycloak KVERSION

Once the service is started you can access the Keycloak Admin Console in order to setup the admin user and perform other administrative tasks
